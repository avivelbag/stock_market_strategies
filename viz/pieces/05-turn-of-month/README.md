# Turn-of-Month Calendar Effect — Visualization

This piece displays the backtest equity curve and regime-conditional Sharpe profile
for the Turn-of-Month Calendar Effect strategy (Lakonishok & Smidt 1988).

The visualization shows ~24% market exposure (5 of every ~21 trading days) and
near-zero performance across all four synthetic datasets, which is the correct
outcome for a calendar-driven effect tested on synthetic data with no institutional
flow mechanism.
