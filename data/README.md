# Synthetic datasets

All CSVs have 1000 business-day rows starting 2020-01-02 and columns
`date, open, high, low, close, volume`.

Generated by `data/generate_synthetic.py` using `numpy.random.default_rng(seed)`.
Re-running the script reproduces byte-identical files.

## Files

### trend_gbm.csv — Geometric Brownian Motion with positive drift
- **Seed**: 42
- **Model**: GBM, mu=0.15/yr, sigma=0.20/yr, S0=100
- **Properties**: Upward-trending with normal return distribution. Baseline
  for any strategy that claims to capture momentum or trend-following alpha.

### mean_rev_ou.csv — Ornstein-Uhlenbeck mean-reversion
- **Seed**: 7
- **Model**: OU process, theta=5/yr, mu=100, sigma=20/yr, S0=100
- **Properties**: Stationary around 100 with moderate oscillation. Tests
  mean-reversion strategies; trend-following should struggle here.

### regime_switch.csv — Alternating trend / mean-reversion regimes
- **Seed**: 123
- **Model**: 200-bar regimes alternating GBM (mu=0.20/yr, sigma=0.20/yr)
  and OU (theta=5/yr anchored to regime-start price, sigma=20/yr)
- **Properties**: Exposes strategies that overfit to a single regime.
  Walk-forward robustness is especially important on this series.

### fat_tail.csv — Student-t(3) fat-tailed innovations
- **Seed**: 999
- **Model**: Zero-drift log-returns with Student-t(3) noise normalized to
  unit variance, sigma=0.20/yr. Price bounded away from zero.
- **Properties**: Extreme daily moves occur ~2–4× more often than GBM
  predicts. Stress-tests stop-loss and position-sizing assumptions.
